## 风控通知

### 日亏损熔断

```
🚨 日亏损熔断
━━━━━━━━━━━━━━
⏰ {时间}
💵 今日亏损: -¥{亏损}
📊 亏损比例: {百分比}%
📝 原因: 日亏损超过 5% 限制
⏸️ 操作: 已暂停交易，请检查策略
📊 当前权益: ¥{权益}
```

### 连续亏损

```
⚠️ 连续亏损预警
━━━━━━━━━━━━━━
⏰ {时间}
📝 连续亏损: {N} 笔
📊 总亏损: -¥{总亏损}
📝 建议: 策略可能失效，请检查市场状态
```


## 日报

```

━━━━━━━━━━━━━━
📊 交易品种: {品种列表}
🔄 今日交易: {N} 笔（{盈利} 盈 {亏损} 亏）
💵 今日盈亏: {+/-}¥{金额}
📊 当前权益: ¥{权益}
📈 累计收益: {+/-}{百分比}%
📉 最大回撤: -{百分比}%
🏆 胜率: {百分比}%
📊 盈亏比: {比值}

📝 今日交易明细:
{逐笔列出: 时间 | 品种 | 方向 | 开仓价 | 平仓价 | 盈亏}

💡 AI 建议:
{根据今日表现给出改进建议}
```

```
📊 盈亏比: {比值}
📉 最大回撤: -{百分比}%

📈 各品种表现:
{品种 | 交易次数 | 胜率 | 盈亏}

💡 本周总结:
{趋势判断、策略表现、改进建议}
```
